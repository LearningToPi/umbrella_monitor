# Release Notes

## What's New

Added an interval in the toml [general] settings.  The script will now run idefinately using the interval provided.

Updated requirements to latests.  pyproject.toml now reflects all dependecies of dependencies and their versions.

Added scripts for building docker cross platform.

Fied issue with the script not terminating with CTRL+C when running the docker container in interactive mode.
